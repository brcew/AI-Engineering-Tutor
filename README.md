# AI Engineering Tutor

A simple AI Engineering Tutor chatbot built using Python and Groq API.

## Features

- AI Engineering Tutor system prompt
- Full conversation history
- Token counting
- JSON conversation saving
- Clean terminal response printing
- Exit using "quit"

## Technologies

- Python
- Groq API
- tiktoken

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python Mini_project.py
```
